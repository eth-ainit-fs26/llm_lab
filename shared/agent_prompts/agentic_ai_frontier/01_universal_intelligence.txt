MATHEMATICAL CONTENT — WHAT YOU MUST INCLUDE
================================================================================

1. SOLOMONOFF INDUCTION
   - Define the universal prior:
       M(x) = Σ_{p : U(p) = x*} 2^{-|p|}
     where U is a universal Turing machine and x* means x is a prefix of U(p).
   - State and prove (or sketch proof of) the convergence theorem:
       Σ_t D_KL( μ(·|x_{<t}) ‖ M(·|x_{<t}) ) ≤ K_U(μ) · ln 2
     i.e., total expected KL divergence is bounded by the Kolmogorov complexity
     of the true environment.
   - Explain why this makes Solomonoff induction the optimal predictor.

2. KOLMOGOROV COMPLEXITY
   - Define K_U(x) = min{ |p| : U(p) = x }.
   - State and prove the Invariance Theorem:
       |K_U(x) − K_V(x)| ≤ c_{U,V}   for all x.
   - Connection to Minimum Description Length (MDL).
   - Why it matters for agents: the simplicity bias is not a heuristic — it is
     the mathematically optimal inductive bias.

3. THE AIXI AGENT — FULL EQUATION
   Write out the AIXI agent:
     a_k = arg max_{a_k} Σ_{o_k r_k} … max_{a_m} Σ_{o_m r_m}
            [r_k + … + r_m] · Σ_{q : U(q, a_{1:k}) = o_{1:k} r_{1:k}} 2^{-|q|}

   Explain each component:
   - The max over actions (agent chooses optimally)
   - The sum over observations/rewards (weighted by Solomonoff mixture)
   - The horizon m
   - Why this is incomputable (reduction to the halting problem)
   State Hutter's self-optimality theorem and Pareto optimality result.

4. GÖDEL MACHINES (Schmidhuber 2003)
   - Setup: the machine has access to its own source code s and an axiomatic
     system A (e.g., ZFC or PA).
   - It searches for proofs of the form:
       A ⊢ "switching from policy s to s' will yield higher expected future reward"
   - If such a proof is found, it rewrites itself.
   - Discuss: Gödelian incompleteness limits what can be proved; the machine
     cannot prove its own consistency; yet the framework is still useful because
     any provable improvement IS implemented.
   - Connection to Löb's theorem and reflective stability.

5. SPEED PRIOR (Schmidhuber 2002)
   - Motivation: Solomonoff prior favours short programs but ignores runtime.
   - Define: S(x) ∝ Σ_{p : U(p)=x} 2^{-K^t(p)} where K^t is time-bounded
     Kolmogorov complexity (penalises both length and runtime).
   - Philosophical implication: predicts the universe is "fast" — computable by
     short, fast programs.

6. COMPUTABLE APPROXIMATIONS RELEVANT TO AGENTIC AI
   - AIXItl: AIXI with time limit t and program-length limit l.
     Convergence guarantees?
   - MC-AIXI-CTW: Monte Carlo tree search + Context Tree Weighting.
     Actually implemented and tested on games (Veness et al. 2011).
   - Feature Reinforcement Learning (Hutter): abstractions over the raw
     observation stream.
   - Connection to modern LLM agents: in what sense is an LLM doing in-context
     learning a crude approximation to Solomonoff induction?

7. OPEN PROBLEMS (at least 5)
   - Can we build computable approximations that preserve AIXI's optimality
     guarantees in meaningful subclasses of environments?
   - The grain-of-truth problem: AIXI assumes the true environment is computable.
     What if it isn't?
   - Self-modification and wireheading: Orseau's results on safely interruptible
     agents.  Can we extend to agents that modify their own reward channel?
   - Is Kolmogorov complexity the right prior for physical-world agents?
     (Alternatives: resource-bounded, logical-depth-based.)
   - Bridging the gap: formal connections between universal AI theory and
     transformer-based agents.
