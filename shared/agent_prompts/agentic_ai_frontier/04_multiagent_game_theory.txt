1. NASH EQUILIBRIUM AND ITS LIMITATIONS
   - Normal-form game: (N, (S_i)_{i∈N}, (u_i)_{i∈N}).
   - Mixed strategy: σ_i ∈ Δ(S_i).
   - Definition: σ* is a Nash equilibrium if
       ∀i, ∀σ_i : u_i(σ*_i, σ*_{−i}) ≥ u_i(σ_i, σ*_{−i}).
   - Nash's existence theorem: sketch proof via Kakutani's fixed-point theorem
     on the best-response correspondence.
   - Limitations:
     (a) Computational hardness: finding a Nash eq. is PPAD-complete (Daskalakis,
         Goldberg, Papadimitriou 2009).
     (b) Equilibrium selection: multiple equilibria, no principled way to choose.
     (c) Beyond 2 players: Nash eq. may not be the right solution concept.
   - Correlated equilibrium (Aumann): define, show it's computable by LP.
   - Coarse correlated equilibrium: the connection to no-regret learning.
     THEOREM: if all players use no-regret algorithms, empirical play converges
     to the set of coarse correlated equilibria.  Prove or sketch proof.

2. COUNTERFACTUAL REGRET MINIMISATION (CFR)
   - Extensive-form game: game tree, information sets, chance nodes.
   - Define counterfactual value:
       v_i(σ, h) = Σ_{z ∈ Z(h)} π^σ_{−i}(h) · π^σ(h,z) · u_i(z)
   - Define counterfactual regret for action a at information set I:
       R^T_i(I, a) = Σ_{t=1}^T [v_i(σ^t_{I→a}, I) − v_i(σ^t, I)]
   - Regret matching: choose actions proportional to positive regret:
       σ^{T+1}(I, a) = R^{T,+}_i(I,a) / Σ_{a'} R^{T,+}_i(I,a')
   - THE KEY THEOREM: In a two-player zero-sum game, if both players use
     CFR, average strategies converge to a Nash equilibrium at rate O(1/√T).
   - Proof sketch: decompose total regret into sum of local counterfactual
     regrets; apply regret matching guarantee at each information set.
   - CFR+ (Tammelin 2014): non-negative regrets only, faster convergence.
   - Deep CFR (Brown et al. 2019): approximate counterfactual values with
     neural networks.  How to handle the massive information set space.

3. SUPERHUMAN GAME PLAY: LIBRATUS AND PLURIBUS
   - Libratus (heads-up no-limit poker):
     - Blueprint strategy via CFR+ on abstracted game.
     - Nested safe subgame solving: solve subgames in real time, guaranteeing
       the refined strategy is no worse than the blueprint.
     - THEOREM (safe subgame solving): if the subgame strategy achieves
       counterfactual value ≥ blueprint at every terminal node of the
       subgame trunk, the overall strategy's exploitability does not increase.
   - Pluribus (6-player poker):
     - Cannot use Nash eq. convergence guarantees (>2 players).
     - Depth-limited search: instead of solving to the end of the game,
       evaluate leaf nodes with the blueprint.
     - Linear CFR: weight recent iterations more heavily.
   - Key insight: the same framework scales from 2 to 6 players, even though
     the theoretical guarantees technically don't hold.

4. LANGUAGE + STRATEGY: CICERO (Meta, Science 2022)
   - The game: Diplomacy — 7 players, simultaneous moves, free-form natural
     language negotiation.
   - Architecture:
     (a) Dialogue model: generates and interprets natural-language messages.
     (b) Strategic reasoning module: piKL (policy regularised to anchor
         towards human-like play, then improved via search).
     (c) Intent model: predicts other players' moves given dialogue.
   - The fusion: the dialogue model proposes messages; the strategic module
     checks whether the messages are consistent with the agent's plan; the
     intent model predicts how others will respond.
   - Signaling games: formalise communication in games.  Define a signaling
     game (sender, receiver, message space, type space).  Babbling equilibrium
     vs. separating equilibrium.  How Cicero avoids babbling.

5. MEAN-FIELD GAMES (Lasry & Lions 2007, Huang, Malhamé & Caines 2006)
   - Setting: N agents, N → ∞, each agent is infinitesimally small.
   - Each agent faces an optimal control problem given the population
     distribution μ_t.
   - Mean-field game equations (the coupled PDE system):
     BACKWARD (Hamilton-Jacobi-Bellman):
       −∂_t V = H(x, ∇V) + F(x, μ_t)
     FORWARD (Fokker-Planck / Kolmogorov):
       ∂_t μ = −∇·(μ ∇_p H(x, ∇V)) + σ²/2 Δμ
   - Mean-field Nash equilibrium: a fixed point where μ_t is consistent with
     agents' optimal behaviour under μ_t.
   - Existence and uniqueness: Lasry-Lions monotonicity condition.
   - Application to multi-agent RL: mean-field MARL (Yang et al. 2018).
     Approximate N-agent interactions by interactions with the mean field.
   - Connection to agentic AI: how to scale multi-agent systems to thousands
     or millions of interacting LLM agents.

6. OPPONENT SHAPING AND LEARNING AWARENESS
   - LOLA (Foerster et al. 2018): gradient through the opponent's learning step.
     Agent 1 updates:
       θ_1 ← θ_1 + α ∇_{θ_1} [V_1(θ_1, θ_2 + δ · ∇_{θ_2} V_2(θ_1, θ_2))]
     i.e., agent 1 accounts for how agent 2 will adapt.
   - Result: LOLA agents learn to cooperate in iterated Prisoner's Dilemma
     where naive learners defect.
   - Model-Free Opponent Shaping (M-FOS, Lu et al. 2022): meta-learning to
     shape opponents without knowing their learning algorithm.
   - Connection to agentic AI: when LLM agents interact repeatedly (e.g.,
     negotiation, debate), should they model how the other agent adapts?
